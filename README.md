## 安装

### 依赖安装

- [LegacyScriptEngine-NodeJS](https://github.com/LiteLDev/LegacyScriptEngine)
- [NodeJS](https://nodejs.org/zh-cn/download)
- [MySQL](https://dev.mysql.com/downloads/mysql/)

### 项目依赖安装

#### 安装 LegacyScriptEngine-NodeJS

在服务端根目录下执行以下命令安装LegacyScriptEngine-NodeJS:

```bash
lip install github.com/LiteLDev/LegacyScriptEngine#nodejs
```
